# 原型文件說明

本資料夾包含智慧機場數位轉型專案的 HTML 原型文件。

## 文件列表

### v1_initial.html

- **建立時間**: 初期開發階段
- **描述**: 第一版 HTML 原型，使用純 HTML/CSS/JavaScript 實現
- **特點**:
  - 基本的系統架構視覺化
  - 簡單的互動功能
  - 展示核心概念

### v2_optimized.html

- **建立時間**: 原型優化階段
- **描述**: 優化版 HTML 原型
- **特點**:
  - 改進的使用者介面
  - 優化的互動體驗
  - 更豐富的視覺效果

## 使用方式

直接在瀏覽器中開啟 HTML 文件即可查看原型：

```bash
# macOS
open v1_initial.html
open v2_optimized.html

# 或使用任何瀏覽器打開
```

## 版本演進

這些原型為 React 版本（`v1/` 資料夾）的基礎，展示了專案從概念驗證到完整應用的演進過程。

## 注意事項

這些文件僅供參考和歷史記錄，當前開發請使用 `v1/` 資料夾中的 React 專案。
